QA Flow is an open-source test automation platform that lets you design, run, and manage Playwright tests through an interactive drag-and-drop canvas. Instead of writing test scripts by hand, you connect nodes representing actions, assertions, and control flow — then execute them with a single click and watch the browser run live.

## Get Started in Minutes

<Steps>
  <Step title="Launch QA Flow">
    Run a single command — Docker is pulled and started automatically:
    ```bash
    npx @davidg97/qa-flow
    ```
    Then open [http://localhost:3001](http://localhost:3001).
  </Step>
  <Step title="Log in">
    Use the default admin credentials: `admin@qaflow.com` / `admin123`. Change these before any production deployment.
  </Step>
  <Step title="Create a project and design your first flow">
    Click **New Project**, drag a **Start** node onto the canvas, set your target URL, then add action and assertion nodes.
  </Step>
  <Step title="Run and review results">
    Click **▶ Execute** and watch the integrated screencast. When the run finishes, open the generated HTML report.
  </Step>
</Steps>

<Note>
  QA Flow runs entirely in your infrastructure — no data leaves your server. The `npx` command pulls the official Docker image and starts a local instance.
</Note>
